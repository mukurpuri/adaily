# Adaily Copy Guidelines

Consistent copy for buttons, states, and messaging across all tools.

---

## üéØ Primary CTAs (10 Options)

Use for main actions. Calm, clear, no pressure.

| CTA | Best For |
|-----|----------|
| **Explore options** | Investment tools |
| **See your results** | After form completion |
| **Check my situation** | Self-assessment tools |
| **Find what fits** | Matching/recommendation tools |
| **Calculate** | Calculators |
| **Show me** | Result reveal |
| **Learn more** | Educational content |
| **View details** | Expanding information |
| **Try this tool** | Tool pages |
| **See how it works** | Onboarding/explanation |

**Avoid:** "Start now", "Get started", "Unlock", "Claim", "Get rich"

---

## üîò Secondary CTAs (10 Options)

Use for alternative actions. Supportive, not competing.

| CTA | Best For |
|-----|----------|
| **‚Üê Back to tools** | Navigation |
| **Explore more tools** | End of tool |
| **Read the guide** | Link to learning content |
| **See other options** | Alternatives |
| **Adjust inputs** | Re-calculation |
| **Compare options** | Side-by-side views |
| **Save for later** | Bookmark functionality |
| **Share results** | Social sharing |
| **View glossary** | Term definitions |
| **Skip for now** | Optional steps |

**Avoid:** "Don't miss out", "Act now", "Before it's too late"

When something goes wrong. Honest, not alarming.

### Option 1: General error

## üí¨ Tone Guidelines

### Do ‚úì
- Use simple words
- Be direct
- Stay calm
- Acknowledge uncertainty
- Educate, don't instruct

### Don't ‚úï
- Create urgency
- Promise outcomes
- Use superlatives (best, fastest, guaranteed)
- Sound like a salesperson
- Pressure action
